## LARAVEL SETUP

- Install composer
- Download php.zip at PHP official website
- Save it in C:php/
- Rename file php.ini-development to php.ini
- Enable comments extension:
    > openssl
    > pdo_mysql
    > zip

- run "composer install" in DIR of imported project
- run "php artisan serve"
